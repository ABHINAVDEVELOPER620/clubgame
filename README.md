A simple Python "High-Low" game where you bet against the computer.

✨ Highlights
Bouncer: Age verification to enter.

Stakes: Place virtual bets to play.

Hints: Tells you to go "Higher" or "Lower."

Limit: 10 tries to win or the house takes it.

🕹️ Quick Start
Run: python ClubGame.py

Pass the age check and place your bet.

Guess the number (1-11) within 10 attempts.

🛠️ Tech
Python 3
